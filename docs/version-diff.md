# Version Difference

Team Control has evolved since 2024, below outlines the major changes in between the code repository. 

# Difference from Team Control 2025

Team Control 2026 is an extended version of 2025 where we focused on delivery with the integration of behaviour tree and rule-based programming. 

# Difference from Team Control 2024.

The old Team operating software utilises a submodule on github known as 'Shared' repository, which stores our main communication protocol that is utilised by both Server and Robot communication. During the competition, this has been troublesome, hence why this has been removed, and GitHub Submodule is no longer a recommendation unless for larger project development.

This year,  we want to focus on getting the robots moving, where we would start from verifying what went wrong and comparing differences between simulation - GRSIM and real-world robots.

We look forward to further utilise the SSL - Game Referee Protocols to simplify the Team controls and hopefully be able to perform basic tasks, movements and complete ball placement. 

With adjustments made, hopefully 2025 - Team Control will be able to exceed our expectations and make our robots move with a more extensive feedback loop.
